## Common Regex Expressions.

# Phone number
# Type
    Phone number
    ((xxx) xxx-xxxx format)

# Pattern
    \(\d\d\d\) \d\d\d-?\d\d\d\d

# Example
    (562) 988-1688
    (562) 9881688

# Social Security Number
# Type
    Social Security number
    (xxx-xx-xxxx)

# Pattern
    \d\d\d-\d\d-\d\d\d\d
    
# Example
    123-45-6789
